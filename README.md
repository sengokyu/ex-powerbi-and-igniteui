# 必要なもの

* powerbi-visuals-tools
* jquery-ui
* Ignite UI

powerbi-visuals-toolsはnpmからインストールします。
jquery-uiとIgnite UIは、ダウンロードして`vendor`ディレクトリ以下に展開してください。

このようなディレクトリ構成になればよいです。

```
vendor/
 + jquery-ui-1.12.1/
 |  + jquery-ui.js
 |
 + ignite-ui/
    + js/
       + infragistics.core.js
       + infragistics.lob.js
```
